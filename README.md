# Mlem Website

[Official website](https://mlem.group) for [Mlem](https://github.com/mlemgroup/mlem), an open source iOS client for the Lemmy.

## Quick Start

```bash
# Install dependencies
npm install

# Development server with Turbopack
npm run dev
```
